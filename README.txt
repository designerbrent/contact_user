User Contact Forms Module
=============================
The User Contact Forms module allows site developers to enable a contact form for certain users by role. This form allows anyone accessing the site to use the form, even anonymous users.

Installation & Setup:
-----------------------------
1: Install the module as usual.
2: Navigate to admin/build/contact/contact_user to configure.
3: Visit admin/user/permissions to select the user roles that you want to have contact forms enabled for.
4: The form is now available at contact/<uid>
